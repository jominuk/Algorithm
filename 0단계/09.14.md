## 숨어있는 숫자의 덧셈

### 문제

```jsx
function solution(my_string) {
    var answer = 0;
    return answer;
}
```

### 문제 풀이

```jsx
function solution(my_string) {
    let answer = my_string.match(/[0-9]/g).reduce((a,c) => {
        return a + parseInt(c) ;
    }, 0)
    
    return answer
}
```
