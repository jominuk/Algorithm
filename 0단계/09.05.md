## 순서쌍의 개수

### 문제

```javascript
function solution(n) {
    var answer = 0;
    return answer;
}
```

### 문제 풀이

```javascript
function solution(n) {
    var answer = 0;
    for(let i = 0;i<=n;i++){
        if(n % i === 0 ){
            answer += 1
        }
    }
    return answer;
}
```
