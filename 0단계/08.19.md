## 배열의 평균값

### 문제 
```javascript
function solution(numbers) {
    var answer = 0;
    return answer;
}
```

### 문제 풀이
```javascript
function solution(numbers) {
    var answer = 0;
    let sum = 0;
    
    for (let i = 0; i < numbers.length; i++){
        sum += numbers[i];
    }
    return answer = sum / numbers.length;
}

// =======================================================

function solution(numbers) {
    return answer = numbers.reduce((n, c) => n + c) / numbers.length
}
```
