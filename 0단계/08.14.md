## 짝수는 싫어요

### 문제 
```javascript
function solution(n) {
    var answer = [];
    return answer;
}
```

### 문제 풀이
```javascript
function solution(n) {
    var answer = [];
    for(let i =0; i <= n; i++) {
        if (i % 2 === 1) answer = [...answer, i]
        
    }
    return answer;
}
```
