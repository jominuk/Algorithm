## 배열 두배 만들기

### 문제 
```javascript
function solution(numbers) {
    var answer = [];
    return answer;
}
```

### 문제 풀이
```javascript
function solution(numbers) {
    let answer = numbers.map((n) =>n*2 )
    return answer;
}

function solution(numbers) {
    let answer = []
    for ( let i =0; i <numbers.length; i++){
        answer[i] = numbers[i]*2
    }
    return answer;
}




```
