## 피자 나눠 먹기 2

### 문제 
```javascript
function solution(n) {
    var answer = 0;
    return answer;
}
```

### 문제 풀이
```javascript
const solution = (n) => {
    let piece = 6

    while(true) {
        if (piece % n === 0) {
            break
        }
        piece += 6
    }

    return piece / 6
}

```
