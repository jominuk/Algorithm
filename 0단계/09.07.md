## 가위 바위 보

### 문제

```javascript
function solution(rsp) {
  var answer = [];
  return answer;
}
```

### 문제 풀이

```javascript
function solution(rsp) {
  var answer = "";
  for (let i = 0; i < rsp.length; i++) {
    if (rsp[i] == "0") {
      answer += "5";
    } else if (rsp[i] == "2") {
      answer += "0";
    } else if (rsp[i] == "5") {
      answer += "2";
    }
  }
  return answer;
}
```
