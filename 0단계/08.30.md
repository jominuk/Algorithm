## 진료 순서 정하기

### 문제

```javascript
function solution(emergency) {
  var answer = [];
  return answer;
}
```

### 문제 풀이

```javascript
function solution(emergency) {
  const temp = [...emergency].sort((a, b) => b - a);
  return emergency.map((v) => temp.indexOf(v) + 1);
}
```
