### 알고리즘 나만의 규칙

- 언어 : `JavaScript`
- 문제 : 프로그래머스 에서 0단계 시작 (최신순 별로 하나씩 풀기)

- 단계별 문제

- 토요일은 꼭 풀이 하고 일요일은 시간 될 경우 최대한으로 ^^

### 마크

- ⭕ 문제 풀었음
- 🔺 어느정도 이해함
- ✖️ 문제 풀지 못함
- ✔️ 못풀었지만 + 이해 O
- 😭 못풀고 + 이해 X
