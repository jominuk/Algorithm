## 숫자 비교하기 

### 문제
```javascript
function solution(num1, num2) {
    var answer = 0;
    return answer;
}
```

### 문제 풀이
```javascript
function solution(num1, num2) {
    var answer = 0;
    if ( num1 === num2) {
         answer = 1
    }else{
        answer = -1
    }
    return answer;
}

function solution(num1, num2) {
    let answer = num1 === num2 ? 1 : -1
     return answer;
}
```
