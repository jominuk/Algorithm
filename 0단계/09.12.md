## 2차원으로 만들기

### 문제

```javascript
function solution(num_list, n) {
    var answer = [[]];
    return answer;
}
```

### 문제 풀이

```javascript
function solution(num_list, n) {
    var answer = [];
    
    for(let i = 0; i < num_list.length; i += n){
        answer.push(num_list.slice(i, i + n))
    }
    return answer;
}
```
