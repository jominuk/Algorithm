### 문제

```javascript
function solution(n) {

}
```

### 문제 풀이
```javascript
function solution(n) {
    var answer = [];
    for(let i =0; i <= n; i++) {
        if (i % 2 === 1) answer = [...answer, i]
        
    }
    return answer;
}
```
